# 🚀 Shefcoin Super-App (AI Bot & Web)

Ин лоиҳа як системаи дутарафа аст, ки аз **Боти Телеграм** ва **Интерфейси Веб** иборат мебошад. Он бо истифода аз AI Gemini кор мекунад.

## 🌟 Имкониятҳо
* **Telegram Bot:** Дорои менюи касбӣ (/start, /profile, /wallet ва ғайра).
* **AI Integration:** Ҷавобҳои ҳушманд ба воситаи Google Gemini.
* **Web Interface:** Намуди зоҳирии сайт дар браузер (Telegram Clone).
* **Backend:** Сохташуда бо Python ва Flask.

## 🛠 Технологияҳо
* **Забон:** Python 3.10+
* **Библиотекаҳо:** `pyTelegramBotAPI`, `Flask`, `google-generativeai`
* **Хостинг:** Render.com

## 📁 Сохтори лоиҳа
* `main.py` — Файли асосии иҷрокунанда (Бот + Сервер).
* `templates/` — Папка барои файлҳои HTML (дизайни сайт).
* `requirements.txt` — Рӯйхати библиотекаҳои лозима.
